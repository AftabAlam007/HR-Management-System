# 🌟 Overview 

**WorkBridge** is an advanced, animated, cloud-ready **HR Management System (HRMS)** built to streamline organization-wide HR operations.

✨ Employee Management  
✨ Payroll & Salary Engine  
✨ Attendance & Leave Workflow  
✨ Recruitment + Onboarding  
✨ Performance Reviews  
✨ Training & Development  
✨ HR Analytics Dashboard  

A seamless integration of **React + TailwindCSS** (frontend) with **Spring Boot + MySQL** (backend).  

---

# ⚡ Tech Stack

## 🎨 Frontend
- React (Vite)
- Tailwind CSS
- Axios
- React Router
- Responsive Layout + UI Animation

## 🛠 Backend
- Spring Boot
- Spring Data JPA
- MySQL
- REST API Architecture
- (Optional: JWT Auth / Role-Based Access)

# 🔐 Authentication + Role Based Access

🔑 Login with **Organization Email + Password**

👥 **HR Dashboard**

# 🗂 Employee Record Classification

---

## 🧍 Personal Details
Includes:
- Name  
- DOB  
- Gender  
- Addresses  
- Mobile  
- Personal Email  
- Emergency Contact  

### Validations:
✔ Valid Email  
✔ 10-digit Mobile  
✔ 6-digit PIN  
✔ Valid Date Formats  

✨ **Employees can update → requires HR approval**  

---

## 🏢 Professional Details
Contains:
- Employee Code  
- Company Email  
- Role  
- Department  
- Reporting Manager  
- HR Manager  
- Date of Joining  

### Restrictions:
❌ Employee Code (locked)  
❌ Company Email (locked)  
❌ DOJ (locked)
